**********Aperçu***********
Ce devoir présente les fondamentaux de JavaScript, un langage de programmation essentiel pour le développement web moderne. À travers cette recherche, j'ai acquis une compréhension solide des concepts de base qui constituent les piliers de toute application JavaScript. Bien qu'il s'agisse de ma première expérience avec ce langage, j'ai pu progresser graduellement en explorant les variables, les fonctions, l'interaction avec le DOM, les méthodes de manipulation de données et la structuration d'objets.

**********Sujets Recherchés**********
 Variables (let, const, var) - Déclaration et portée des variables

 Fonctions - Création et utilisation de fonctions

 Manipulation du DOM - Interaction avec les éléments HTML

 Méthodes des Tableaux - Traitement et itération sur les données

 Objets & JSON - Structure et sérialisation des données

**********Apprentissages Clés**********
Distinction entre let, const et var - En commençant par JavaScript, j'ai découvert que ces trois mots-clés déclarent des variables, mais de manières différentes. "const" est idéal pour les valeurs invariable, "let" offre une portée de bloc bien définie, tandis que "var" représente l'ancienne approche. Comprendre ces différences m'a aidé à écrire un code plus robuste et à éviter des erreurs subtiles liées à la portée.

Les fonctions sont des outils puissants pour organiser le code et l'utiliser plusieurs fois sans répétition. J'ai compris comment declarer des paramètres, récupérer des valeurs avec return, et comment les fonctions structurent logiquement un programme. C'est un concept fondamental qui me permet de créer du code plus propre et maintenable.

Le DOM joue un rôle d'une interface interactive: JavaScript me permet de communiquer directement avec les éléments HTML et de les modifier en temps réel. Grâce aux sélecteurs et aux événements, j'ai pu créer des interactions dynamiques. Par exemple, cliquer sur un bouton peut maintenant déclencher une action, ce qui rend les pages web vivantes et réactives.

Les tableaux et leurs méthodes puissantes comme map(), filter(), et forEach() facilite le traitement des données de manière plus concise. Ces méthodes remplacent les boucles traditionnelles et rendent le code plus lisible et efficace.

Les objets permettent de structurer les données de façon logique en regroupant des propriétés et des méthodes. Découvrir JSON (JavaScript Object Notation) a montré comment ces structures peuvent être facilement converties en texte pour être stockées.

**********Difficultés Rencontrées**********
Au début, comprendre la portée des variables (scope) a été un défi, notamment la différence entre la portée globale et la portée de bloc. J'ai surmonté cette difficulté en pratiquant avec des exemples simples et en testant dans la console du navigateur pour visualiser le comportement.

Une autre difficulté a été de bien maîtriser la syntaxe des fonctions fléchées versus les fonctions traditionnelles. En relisant la documentation et en comparant les deux syntaxes dans des projets pratiques, j'ai progressivement compris leurs différences et leurs cas d'utilisation appropriés.

La manipulation du DOM semblait intimidante au départ, mais en travaillant avec des éléments individuels et en testant les modifications en temps réel, j'ai développé une confiance croissante. Les erreurs console m'ont guidé vers les bonnes solutions.

Enfin, bien que JSON soit un concept simple une fois compris, la sérialisation et désérialisation avec JSON.stringify() et JSON.parse() ont nécessité quelques essais avant de devenir maniable.
